## Были реализованы тесты с помощью Vitest и Playwright:
### 1. Добавление в корзину (unit)
Тестирует стор Корзины, содержит 4 теста (добавление продукта, добавление уже добавленного продукта, уменьшение продукта при количестве больше/равному 2, уменьшение при количестве равному 1).
### 2. Проверка логина (unit)
Тестирует компонент Login, содержит два теста (попытка аунтификации с неверными данными и попытка с верными).
### 3. Загрузка данных (unit)
Тестирует стор Продуктов, содержит 2 теста (загрузка данных и вывод "Загрузка..." при отсутствии загруженных данных).
### 4. Расчет суммы в корзине, с изменениями количества товара (e2e)
Тестирует приложение на добавление товаров в корзину, добавление и убавление товаров в корзине и проверку общей суммы на каждом этапе.
