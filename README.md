## Были реализованы компоненты:
### 1. Меню навгации
Элемент Вход/Выход зависит от "авторизованности".
### 2. Список товаров в корзине
Корзина прокинута единым массивом айдишников через props и emits (хранится в App.vue). 
На каждом товаре кнопки "Добавить" и "Уменьшить", увеличивающая и уменьшающая продукт в корзине на 1 соответственное. По достижении 0 продукт удаляется. Кнопка "Купить" вызывает форму чекаута, после её заполнения продукты из корзины удаляются. Кнопка "Очистить" очищает корзину.
### 3. Главная страница
Кнопка добавления товара появляется при "авторизации". На каждом товаре кнопка "В корзину", увеличивающая продукт в корзине на 1.
### 4. Страница товара
Айди передаётся через параметр, данные о продуктах прокинуты массивом через props и emits (хранится в App.vue).

## Navigation guards
Был использован befrEach для перенаправления пользователей в случае:
1. Переход на форму добавления продукта, но пользователь не админ => форма входа
2. Переход на форму входа, но пользователь авторизован => главная
