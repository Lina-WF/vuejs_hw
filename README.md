## Были реализованы сторы с помощью Pinia:
### 1. Стор товаров
Содержит загруженные из API товары, а так же действия с ними: проверка информации, цены о товаре, добавление и поиск товара.
### 2. Стор корзины
Содержит список айдишников и количества в корзине, а так же список продуктов, отфильтрованный из всех продуктов по списку айдишников. Содержит такие действия как увеличение, уменьшение количества в корзине, очистка корзины, проверка корзины и общая сумма корзины.
### 3. Стор пользователя
Содержит объект с данными об администраторе и два действия: вход и выход. Записывается и читается из localStorage.
### 4. Стор фильтра
Содержит поля фильтра и отфильтрованные из всех продуктов по этому фильру. Действиями являются очистка фильтра и установка новых значений.
### 5. Стор фильтрованых продуктов
Содержит отфильтрованные из всех продуктов по фильру.
